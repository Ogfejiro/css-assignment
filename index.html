MARGIN:The CSS margin properties are used to create space around elements, outside of any defined borders. With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom, and left).

PADDING:The CSS padding properties are used to generate space around an element's content, inside of any defined borders.There are properties for setting the padding for each side of an element (top, right, bottom, and left).

BORDER:The CSS border properties allow you to specify the style, width, and color of an element's border.
